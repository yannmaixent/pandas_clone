# pandas_clone

-   Understand Pandas internal architecture
-   Rebuild core APIs from scratch
-   Focus on correctness, clarity, and learning

## Roadmap

-   v0.2:   DataFrame
-   v0.3:   Indexing (.loc/.iloc)
-   v0.4:   groupby, merge, CSV IO
